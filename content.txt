New Slide Title: Dual-Layer Architecture — Our Key Innovation
Subtitle (italic, grey):
Decouples UI responsiveness from the computationally heavy multi-agent pipeline.
Layout: Two columns side by side
🟢 LEFT BOX — SYNCHRONOUS LAYER (green border)
Tagline: Fast • < 1 second response • Keeps UI live

Rule-based deterministic difficulty mapping
Fixed transition pattern:
Easy → Medium → Hard → Hard → Hard → Medium → Easy → Easy
Returns next question immediately in JSON response
No LLM call on the critical UI path
Guarantees real-time question selection

🔵 RIGHT BOX — ASYNCHRONOUS LAYER (blue border)
Tagline: Deep AI reasoning • Runs in background daemon thread

Full CrewAI pipeline runs after answer submission:
MCQ Evaluator → Difficulty Predictor → Overseer
Powered by Gemini 2.5 Flash via CrewAI
AI analysis persisted to AdaptiveLearningHistory
Updates final_difficulty after deep reasoning
DB connection closed in finally block — no leaks

Bottom Callout (light green box, centered, bold italic):
💡 "User never waits for the LLM — but every decision is still backed by deep AI reasoning."
